# UAV Telemetry Parameters Reference

This document describes the telemetry parameters generated by the `uav-simulation` mock script. These parameters are designed to align with standard **MAVLink** messages used in real flight controllers (like ArduPilot or PX4) to simplify the future migration to `node-mavlink`.

## 📍 Navigation & Position
| Parameter | Description | MAVLink Alignment |
| :--- | :--- | :--- |
| `lat` | **Latitude**: Current GPS coordinate. | `GLOBAL_POSITION_INT` |
| `lon` | **Longitude**: Current GPS coordinate. | `GLOBAL_POSITION_INT` |
| `alt_rel` | **Relative Altitude**: Height in meters above the takeoff point. | `GLOBAL_POSITION_INT` / `VFR_HUD` |
| `heading` | **Compass Heading**: Direction the nose is pointing (0° North, 90° East). | `VFR_HUD` / `ATTITUDE` |
| `fix_type` | **GPS Health**: `3` = 3D Fix (Good), `<3` = Unreliable/Lost. | `GPS_RAW_INT` |

## ✈️ Flight Dynamics
| Parameter | Description | MAVLink Alignment |
| :--- | :--- | :--- |
| `airspeed` | **Airspeed**: Speed of the aircraft relative to the air (m/s). | `VFR_HUD` |
| `groundspeed` | **Groundspeed**: Speed of the aircraft relative to the ground (m/s). | `VFR_HUD` |
| `pitch` | **Pitch**: Nose up (+) or nose down (-) angle in degrees. | `ATTITUDE` |
| `roll` | **Roll**: Left lean (-) or right lean (+) angle in degrees. | `ATTITUDE` |
| `throttle` | **Throttle**: Motor power percentage (0–100%). | `VFR_HUD` |

## 🔋 Diagnostics & Health
| Parameter | Description | MAVLink Alignment |
| :--- | :--- | :--- |
| `batt_rem` | **Battery Remaining**: Percentage of battery left (0–100%). | `SYS_STATUS` |
| `servo_current`| **Servo Current**: Electrical current (Amps) used by actuators. | `SYS_STATUS` |
| `vibration` | **Vibration Vector**: `[X, Y, Z]` physical shaking in m/s². | `VIBRATION` |
| `rssi` | **Signal Strength**: Strength of the RC link (0–254). | `RC_CHANNELS` |

## 🎥 Mission & Status
| Parameter | Description | Notes |
| :--- | :--- | :--- |
| `cam_status` | Camera state: `ACTIVE`, `DEGRADED`, or `OFF`. | Custom extension |
| `signal_quality`| Video/Data link quality percentage (0–100%). | Custom extension |
| `loiter_radius` | Radius of the circle (m) if the drone is orbiting. | `NAV_CONTROLLER_OUTPUT` |

## 🕒 Metadata
| Parameter | Description |
| :--- | :--- |
| `timestamp` | ISO 8601 formatted time of packet generation. |
| `checksum` | XOR-based integrity check to ensure data wasn't corrupted. |
